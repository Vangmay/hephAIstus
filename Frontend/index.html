What is HephAIstos?

HephAIstos is an autonomous coding assistant that leverages multiple AI APIs to help you write, review, and refactor code effortlessly.

Features

  • Multi-API support (Groq, Exa, OpenAI)
  • Environment-based configuration
  • Interactive CLI
  • Graceful error handling

Quick Setup

  1. Clone the repo
  2. Install deps: pip install -r requirements.txt
  3. Create .env with your API keys
  4. Run python main.py
